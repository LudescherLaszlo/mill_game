# Nine Men's Morris (Mill) :

A Python implementation of the classic strategy board game [Nine Men's Morris](https://en.wikipedia.org/wiki/Nine_men%27s_morris). Challenge the computer in a battle of wits, form "mills" to remove your opponent's pieces, and reduce them to two pieces or block all their legal moves to win!

## ✨ Features

* **Human vs. Computer Gameplay:** Test your strategic skills against a computer opponent.
* **Smart AI:** The computer employs strategic gameplay—it actively looks for immediate winning moves and will intelligently block your attempts at a 1-move victory. *(If you implemented the Minimax bonus, you can mention: "Powered by a Minimax algorithm for highly competitive play.")*
* **Robust Input Validation:** The game engine fully protects itself against invalid user inputs, ensuring a smooth and crash-free experience.
* **Dual Interfaces:** *(If you implemented the GUI bonus, include this)* Play via a classic text-based Console UI or a Graphical User Interface (GUI), both seamlessly integrating with the core game engine.

## 🏗 Architecture & Technologies

This project was built with clean code principles in mind:
* **Language:** Python
* **Architecture:** Strict Layered Architecture separating the User Interface, Business Logic/Services, and Data layers.
* **Design:** Object-Oriented Programming (OOP) principles utilized throughout the codebase.

## 🧪 Testing

The core game logic and non-trivial methods are thoroughly tested using **PyUnit** to ensure the rules of Nine Men's Morris are accurately enforced and edge cases are handled.
